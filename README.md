# CoWatch — landing page

Static landing page for CoWatch (the household watch tracker), served via GitHub Pages.
Single self-contained `index.html` (no build step).

## Deploy
1. This repo must be **public** (GitHub Pages free tier).
2. Settings → Pages → Source: **Deploy from a branch** → `main` / `/ (root)` → Save.
3. Live in ~1 min at the URL shown there.

## To collect waitlist emails
The form posts to Formspree — create a free form at https://formspree.io and replace
`REPLACE_ME` in `index.html` with your form id. Until then the form is a placeholder.

App source lives in a separate private repo.
